Number Checker (C#)

A simple console application that checks whether an entered integer is:

- Less than 20  
- Exactly 20  
- Greater than 20  

## How to Run

1. Install **.NET SDK (version 8.0 or newer)**  
2. Open terminal inside project folder  
3. Run:

```
dotnet run
```

## Project Structure

```
CSharpNumberChecker/
│── NumberChecker.csproj
│── Program.cs
│── README.md
```

## Example

Input:
```
15
```

Output:
```
The number is less than 20.
```
